# Praktikum Semaphore & Mutex 

Repository ini menyediakan source code Implementasi **Semaphore & Mutex** 

---

## Installation
Buka terminal di VM Xinu, lalu jalankan pada directory xinu:
1. **Clone File signaling.c:**
    ```bash
    wget https://raw.githubusercontent.com/ramandarafky/semaphore-implementation/refs/heads/main/signaling.c
    ```
2. **Clone File mutex.c:**
    ```bash
    wget https://raw.githubusercontent.com/ramandarafky/semaphore-implementation/refs/heads/main/mutex.c
    ```
